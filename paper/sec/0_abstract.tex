\begin{abstract}
Street-view perception models can predict subjective attributes such as safety and wealth at scale, yet they rarely answer the question practitioners care about: for this specific street, which visual changes would plausibly shift perception? We propose a lever-based interventional attribution framework for street-view perception. Rather than treating arbitrary image regions as the explanatory unit, we define structured lever interventions - grounding each candidate in a semantic concept, scene support, intervention direction, and constrained edit template - and instantiate a closed scene-specific candidate set from a curated ontology of manipulable urban cues. For each image, we generate bounded prompt-only edits, audit them for same-place preservation, locality, realism, and plausibility, and retain only accepted interventions for downstream evaluation. We define the observed effective lever set $E(x, a)$ as the subset of accepted candidate interventions that exceed a preregistered human pairwise preference threshold under single-lever testing. On an N=30 pilot from the SPECS dataset (Singapore subset), we use generator-validity diagnostics and auxiliary classifier scoring to prioritise candidate interventions while positioning randomized human pairwise evaluation as the primary endpoint. This reframes interpretability for urban perception as a human-grounded protocol for testing scene-specific visual hypotheses rather than inferring full causal structure.


\end{abstract}
