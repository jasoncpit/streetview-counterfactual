This directory is a self-contained arXiv submission copy of `paper/`.

Notes:
- `main.tex` uses `\usepackage[pagenumbers]{cvpr}` for a compact single-spaced arXiv layout.
- `main.bbl` is included so arXiv can compile references without depending on a local BibTeX run.
- Only the files needed by `main.tex` should be uploaded.

Recommended upload contents:
- `main.tex`
- `preamble.tex`
- `cvpr.sty`
- `ieeenat_fullname.bst`
- `main.bib`
- `main.bbl`
- `sec/*.tex`
- `figures/*.png`
